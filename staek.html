Pengantar Analisis Korelasi & Regresi Linear
🎯 Tujuan Tutorial Tutorial ini akan mengajarkan cara membuat analisis korelasi dan regresi linear sederhana menggunakan data penjualan 100 produk makanan ringan dari Shopee. Kita akan menggunakan Microsoft Excel dari awal hingga interpretasi hasil.

📐 Konsep Utama
Korelasi Pearson (r): Mengukur kekuatan hubungan linear antara X (penjualan) dan Y (pendapatan). Nilai -1 hingga +1.
Regresi Linear: Membuat persamaan Ŷ = a + bX untuk memprediksi pendapatan berdasarkan penjualan.
Koefisien Determinasi (R²): Persentase variasi Y yang dapat dijelaskan oleh X.
✅ Hasil Akhir yang Akan Diperoleh Persamaan regresi Ŷ = 5.280 + 2.541X dengan korelasi r = 0.97 (sangat kuat) dan R² = 0.94 (94% variasi pendapatan dijelaskan oleh penjualan).

Menemukan persamaan Ŷ = a + bX

🎉 Persamaan Regresi: Ŷ = 5.28 + 2.54X
  • → a = 5.28: Jika penjualan = 0, pendapatan diperkirakan Rp 5.28 juta (nilai dasar)
  • → b = 2.54: Setiap kenaikan 1 unit penjualan meningkatkan pendapatan Rp 2.54 juta
📐 Formula Excel =SLOPE(D2:D101, C2:C101) → b = 2.5408 (Y dulu, baru X!) =INTERCEPT(D2:D101, C2:C101) → a = 5.2808 =FORECAST(X_baru, D2:D101, C2:C101) → Prediksi Y
💡 Cara Menambah Kolom Ŷ (Prediksi) Di kolom E header: ketik Ŷ (Prediksi Y), lalu di E2 ketik: =$F$20 + $F$18 * C2 lalu drag ke bawah hingga E101. Ini menghasilkan nilai prediksi untuk setiap produk.
